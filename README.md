# ClearCarGo

## Installation

- `pnpm install`

## Environment Variables

### Required

- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- SUPABASE_SERVICE_ROLE_KEY
- NEXT_PUBLIC_FRONT_END_URL

### Sending Emails

- SMTP_HOST
- SMTP_PORT
- SMTP_USER
- SMTP_PASS
- SMTP_EMAIL
- SMTP_NAME

### Environment

- NODE_ENV
